## Инструкция по сборке
Чтобы сбилдить проект
```
make build
```
Чтобы установить модуль в систему
```
make install
```
Чтобы удалить драйвер
```
make remove
```
Чтобы отчистить директорий от файлов сборки
```
make clean
```
